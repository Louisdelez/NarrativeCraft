# NarrativeCraft Tutorial Examples

Welcome to the NarrativeCraft tutorial! These example stories will teach you how to create interactive narratives in Minecraft.

## Tutorials

### 01 - First Dialog
**File:** `01-first-dialog.ink`

Learn the basics:
- Scene structure with `# on enter`
- Writing dialog (`Character: Text`)
- Fade transitions (`# fade`)
- Saving progress (`# save`)

### 02 - Choices
**File:** `02-choices.ink`

Create branching narratives:
- Player choices (`* [Choice text]`)
- Variables (`VAR name = value`)
- Conditional content (`{ condition: ... }`)
- Story organization (knots and stitches)

### 03 - Cutscenes
**File:** `03-cutscene.ink`

Add visual effects:
- Cinematic borders (`# border`)
- Screen shake (`# shake`)
- Sound effects (`# sfx`, `# song`)
- Time and weather control
- Camera angles
- Combining effects

## How to Use

1. **Copy the file** into your NarrativeCraft project
2. **Create a scene** in the NarrativeCraft editor
3. **Paste the Ink code** into the scene's script file
4. **Build and play** the story

## Prerequisites

- NarrativeCraft mod installed
- Basic familiarity with Minecraft
- For tutorial 03: Camera angles set up in the editor

## Learning Path

1. Start with `01-first-dialog.ink`
2. Progress to `02-choices.ink`
3. Complete with `03-cutscene.ink`
4. Study the advanced showcase: `../showcase/complex-scene.ink`
5. Create your own stories!
